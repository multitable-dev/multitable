# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in MultiTable, please report it responsibly.

**Do NOT open a public issue.**

Instead, email **security@multitable.dev** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.x.x   | Yes (current development) |

## Scope

This policy covers the MultiTable application and CLI tool. Third-party dependencies should be reported to their respective maintainers.
